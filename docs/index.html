9 Tools, One MCP Server

Everything you need to design a complete electrical system — from diagrams to cable sizing to battery configuration.

Wiring Diagram Generator

Generate complete SVG or PNG schematics with automatic protection components — shunt, main switch, and low-voltage cutoff.

Power Budget Calculator

Calculate daily energy consumption from a list of loads with watt-hours, peak draw, and runtime analysis.

Battery Bank Sizing

Determine capacity, number of batteries, and series/parallel configuration with depth-of-discharge and autonomy settings.

Solar Panel Sizing

Calculate required solar wattage based on daily consumption, location sun hours, and system efficiency.

Charging Time Estimator

Estimate charge duration from solar, shore power, alternator, wind, and generator with bulk/absorption phases.

Inverter Sizing

Size AC inverters for DC battery systems with continuous load, surge handling, and efficiency calculations.

Battery Configuration

Determine series/parallel arrangements with step-by-step wiring instructions and terminal connection guides.

Cable Cross-Section Calculator

Find optimal wire gauge with ampacity, voltage drop analysis, and fuse recommendations for any DC circuit.

Component Library

Browse all available component types — panels, batteries, chargers, loads, and protection devices.

Quick Start

Connect to the MCP server in seconds. Choose the method that fits your workflow.

claude_desktop_config.json
{ "mcpServers": { "voltplan-wiring": { "type": "url", "url": "https://mcp.voltplan.app/mcp" } } }
Terminal
# Add as remote MCP server claude mcp add voltplan-wiring \ --transport http \ https://mcp.voltplan.app/mcp
Terminal
# Run locally via npx (no install needed) npx wiring-diagram-mcp # Or start as HTTP server npx wiring-diagram-mcp --http
Terminal
# Build and run with Docker docker build -t wiring-diagram-mcp . docker run -p 3001:3001 wiring-diagram-mcp
