# Examples

Five worked scenarios. Each file contains the setup, the prompt to give Claude, the tools Claude should call, and the resulting numbers.

| Scenario | System | Daily use | Bank | Solar |
|---|---|---|---|---|
| [Sailboat refit](./sailboat-refit.md) | 12V | ~2 kWh | 500 Ah LiFePO4 | 600 W |
| [Sprinter van](./sprinter-van.md) | 12V | ~1.8 kWh | 400 Ah LiFePO4 | 600 W + alternator |
| [Off-grid cabin](./off-grid-cabin.md) | 24V | ~4.4 kWh | 800 Ah @ 24V | 3 kW |
| [Day sailer](./day-sailer.md) | 12V | ~180 Wh | 100 Ah AGM | 100 W |
| [Live-aboard](./live-aboard.md) | 24V | ~8.6 kWh | 800 Ah @ 24V | 4 kW + shore |

All numbers come from the MCP's calculator tools — drop the prompt into Claude with the server connected and reproduce.
